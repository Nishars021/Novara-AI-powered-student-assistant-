🚀 Novara – AI Powered Personalized College Assistant

Novara is an AI-powered personalized student assistant designed to help college students improve academic performance, manage studies efficiently, and achieve career goals through intelligent insights and personalized guidance.

Unlike traditional chatbots, Novara analyzes student data such as marks, academic performance, study habits, and goals to provide smart recommendations, study planning, CGPA prediction, and career guidance.

✨ Features
📊 CGPA Predictor : Predicts expected CGPA,Calculates required marks to achieve target CGPA

📚 Smart Study Planner :P Generates personalized study schedules,Prioritizes weak subjects and upcoming exams

🤖 AI Assistant : Answers academic and college-related queries,Provides personalized recommendations

📈 Performance Analyzer : Tracks academic progress,Identifies weak areas and improvement opportunities

🎯 Career Guidance : Suggests career paths and required skills,Provides learning roadmaps

🔔 Smart Notifications : Exam reminders,Study recommendations,Productivity alerts


🧠 What Makes Novara Different?

Unlike normal chatbots, Novara acts as a student intelligence system by:

Using personalized academic data,
Providing decision-based recommendations,
Combining study planning, analytics, and career guidance into one platform

🛠️ Tech Stack

Frontend : HTML,CSS,JavaScript

Backend : Python,Flask

Database : Firebase / MongoDB

AI & ML :Scikit-learn,Rule-based recommendation system

🎯 Project Goal

To create a smart digital academic companion that helps students:
